My favorite part of this course has been the applicability of the technical skills we are being taught. I have been tasked in the past with designing web pages, so now that I understand HTML and CSS I can create one from scratch.

Having a professor who is a working professional and interacts with systems every day has made this class truly rewarding. Learning about the process of creating and implementing an information system has been an invaluable skill to add to my tool box.

This course has raised my awareness about the historical ways in which public perceptions of race and identity have been formed.
